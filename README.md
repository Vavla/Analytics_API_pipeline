# Задание 2 — API-пайплайн: данные → LLM → результат  

## Цель:  
Написать рабочий скрипт, который автоматически отправляет данные в LLM через API и получает структурированный ответ.  

## Инструкция к запуску:  
1. Клонировать файлы из репозитория
2. Переименовать файл .env.example в .env, вместо REPLACE_YOUR_KEY вставить свой ключ для OPENROUTER (для начала стоит зарегистироваться в этом сервисе)  
3. Установить библиотеки  csv, json, pandas, openai, dotenv через команду `pip install`, если они ранее не были установлены.  
4. Запустить программу через терминал (Например, `py -3.13 main.py`) либо кнопку запуска.

## Требования: 
1. В входном файле input.csv должны находиться данные о товарах в формате (пример записи ниже):  
    ```
    id,product,description  
    ...
    4,Футболка CottonWear,"Бренд: StyleMe, Цена: 990 руб. Хлопковая футболка классического кроя. Дышащий материал, не теряет форму после стирок."  
    ...  
    ```    

2. В выходном файле все данные записываются в формате JSON, файл должен быть создан в той же директории, где лежит проект (это касается и входного файла).  
3. Формат выходного файла (пример ниже):  
    ```
    {
    "id": "1",  
    "price": "24990",  
    "brand": "TechPro",  
    "category": "Электроника"  
    },  
    {  
        "id": "2",  
        "price": "4590",  
        "brand": "SportMax",  
        "category": "Обувь"  
    },  
    ```
